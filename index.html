Grassroots Global Justice

Website

The Grassroots Global Justice Alliance (GGJ) is a national alliance of US-based grassroots organizing (GRO) groups organizing to build an agenda for power for working and poor people and communities of color. In 2015 I build a new theme for the GGJ's Drupal 7 website, which pulls in social media feeds from Twitter, Facebook and Wordpress.

Action Network Plugin

Wordpress Plugin

A Wordpress plugin for the Action Network online organizing tools. Allows users to:

  • Create a Wordpress shortcode from any Action Network embed code
  • Manage saved embed codes using the Wordpress backend
  • Use API keys to sync actions from Action Network to Wordpress
  • Provides a shortcode and Action Network Calendar widget to show a list of upcoming events
